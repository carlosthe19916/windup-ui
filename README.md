# windup-ui

> Windup UI

[![NPM](https://img.shields.io/npm/v/windup-ui.svg)](https://www.npmjs.com/package/windup-ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save windup-ui
```

## Usage

```tsx
import * as React from 'react'

import MyComponent from 'windup-ui'

class Example extends React.Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
